# v0.5.0

This release introduces Horizontal Federated Analytics (HFA) task. Developers could write analytics tasks using Pandas, and run the task on the multi-node Delta network. The Pandas code is executed locally on each node, a horizontal secure aggregation is performed whenever cross node data aggregation is required by one of the Pandas operation. There will be multiple cross node secure aggregations during the execution of a single Delta HFA task. The original data and single-node-aggregation result remain private for each node during the whole execution process.
